# Data Notes

The dataset used in this project comes from the coursework case study for a fashion accessories retailer supplier-selection problem.

The analysis uses supplier-level data for 12 anonymised suppliers labelled A to L.

Data categories include:

- pricing and total purchase value,
- product quality indicators,
- delivery performance indicators,
- customer service indicators,
- qualitative survey-based belief distributions.

The data is embedded inside `src/data/loader.py` so the project can run without requiring external files.
